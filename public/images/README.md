78장 전체 카드 이미지가 이 폴더에 들어 있습니다.

출처: Pamela Colman Smith가 그린 1909년 Rider–Waite–Smith 덱 스캔본으로,
[Wikimedia Commons](https://commons.wikimedia.org/wiki/Category:Rider-Waite_tarot_deck)에서
가져왔습니다 (미국 저작권 만료로 퍼블릭 도메인).

파일명 규칙은 `src/data/cards.json`의 각 카드 `image` 필드와 일치해야 합니다.
예: `major-00-fool.jpg`

이미지가 없으면 앱이 자동으로 슈트별 색상 플레이스홀더를 대신 보여줍니다.
